- **2nd April 2024**: Challenge launch with dataset and baseline software (Task 1 and 2)
- **8th May 2024**: Task 3 dataset released
- **25th July 2024**: Evaluation data released
- ~~2nd~~ **9th Sept 2024**: First round submission deadline for evaluation by objective measure

- ~~Sept~~ **9th Sept 16th**:  Evaluation data for listening tests released to top teams

- ~~16th~~ **23rd Sept 2024**: Second round submission deadline for listening tests

- **Oct-Nov 2024**: Listening test evaluation period.
- **5th Dec 2024**:  Results announced at a Clarity Challenge Workshop; prizes awarded. The workshop will be a one-day virtual event.
