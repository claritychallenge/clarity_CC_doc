## Learn more about the challenge

To support the third Clarity Enhancement Challenge for Hearing Aid Signal Processing we’re running a webinar on the 10th May 2024. It will be run twice to help people to join from different time zones.

In the webinar the Clarity team will outline the challenges, rules, data, baseline and tools available. We will then run a Q&A. Everyone is welcome to join. The sessions will also be recorded and made available online afterwards for anyone that can’t attend the live event.
